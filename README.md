# Java Swing Calculator

This project is a simple calculator built using **Java Swing**.  
It is an extract from a larger learning repository called **my-java-learning-journey**.

The main goal of this project is to practice and understand **event handling in Java**, especially using `ActionListener` and GUI components.

---

## Purpose

This calculator was created as part of my learning process in Java GUI programming.  
It focuses on:

- Handling button events with `ActionListener`
- Managing application state (current result and last operation)
- Building interfaces using `JFrame`, `JPanel`, `GridLayout`, and `BorderLayout`
- Connecting user input with application logic

---

## What I learned

- How to use multiple `ActionListener` classes for different types of buttons
- How to process user input from GUI components
- How to store and apply the last selected operation (`+`, `-`, `*`, `/`, `=`)
- How event-driven programming works in Java Swing

Special emphasis was placed on **event-driven logic**, understanding how each button triggers actions and how the program reacts to them.

---

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Graphical interface with buttons and display
- Continuous calculations using stored state
- Decimal number input
